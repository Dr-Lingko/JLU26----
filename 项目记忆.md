## 第一问建模目标

第一问要计算每条路段在不同时刻的：

```text
积水深度 h_i(k)
通行能力系数 K_i(k)
安全度评分 S_i(k)
```

核心链条为：

```text
降雨过程 + 路段基础条件 + 排水能力
        ↓
积水深度 h_i(k)
        ↓
通行能力系数 K_i(k)
        ↓
安全度评分 S_i(k)
```

## 第一问最终采用的模型

经过讨论，第一问不要使用纯查表法，也不要主观设线性权重。最终采用：

> 基于工程排水关系的路段积水动态模型，并利用附表 4 进行最小二乘参数校准。

正式公式为：

```latex
\begin{equation}
h_i(k)=\max\left\{0,\,
h_i(k-1)
+\alpha R_k \psi_i L_i(1+\lambda E_i)
-\beta Q_i'
-\gamma \sqrt{G_i}
\right\}
\end{equation}
```

其中：

- `h_i(k)`：第 i 条路段在第 k 个时刻的积水深度，单位 cm。
- `R_k`：第 k 个时段的降雨量，来自附表 2。
- `psi_i`：路面径流系数，由路面类型确定。
- `L_i`：路段长度归一化值。
- `E_i`：低洼程度，由最低标高计算。
- `Q_i'`：设计排水能力归一化值。
- `G_i`：地形坡度归一化值。
- `alpha, beta, gamma, lambda`：待校准参数。

各归一化变量为：

```latex
\begin{equation}
E_i=\frac{Z_{\max}-Z_i}{Z_{\max}-Z_{\min}},\quad
L_i=\frac{l_i}{l_{\max}},\quad
Q_i'=\frac{Q_i}{Q_{\max}},\quad
G_i=\frac{g_i}{g_{\max}}
\end{equation}
```

## 第一问公式依据

该公式的结构不是随便设定，来源于实际排水工程关系：

1. 降雨径流部分参考推理公式法：

```text
Q = psi * q * F
```

即雨水设计流量与暴雨强度、径流系数、汇水面积近似成正比。

2. 管网和道路排水部分参考 Manning 公式：

```text
Q = (1/n) * A * R_h^(2/3) * S^(1/2)
```

说明排水能力与坡度常呈平方根关系，因此模型中使用 `sqrt(G_i)` 表示坡度自然排泄作用。

3. 最低标高不直接出现在简化排水公式中，但反映低洼汇水倾向，因此构造：

```text
E_i = (Zmax - Zi) / (Zmax - Zmin)
```

`E_i` 越大，说明路段越低洼，越容易积水。

## 参数计算方法

参数不能主观设定，应使用附表 4 的积水深度数据进行校准。

附表 4 给出 L1-L8 在：

```text
15 min, 30 min, 45 min, 60 min
```

的积水深度，记为 `H_i(k)`。

建立目标函数：

```latex
\begin{equation}
\min_{\alpha,\beta,\gamma,\lambda}
J=\sum_{i=1}^{8}\sum_{k=1}^{4}
\left[h_i(k)-H_i(k)\right]^2
\end{equation}
```

约束条件建议为：

```latex
\alpha\ge 0,\quad \beta\ge 0,\quad \gamma\ge 0,\quad \lambda\ge 0
```

算法名称：

```text
约束最小二乘参数估计
```

MATLAB 可用：

```text
lsqnonlin
fmincon
```

如果暂时做简化，也可使用非负最小二乘 `lsqnonneg`。

## 路面径流系数设置

路面径流系数应参考工程取值，不是主观权重。

根据排水规范中混凝土或沥青路面径流系数通常较高的原则，本题可设置：

```text
沥青路面：psi_i = 0.95
水泥路面：psi_i = 0.90
```

如果论文中希望更保守，也可统一说明：

```text
由于题目路面均为硬化路面，径流系数取 0.90-0.95。
```
